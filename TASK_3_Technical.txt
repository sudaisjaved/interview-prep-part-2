    RETURN "Error: Password must be at least 8 characters"

  IF password does not contain any digit (0 through 9):
    RETURN "Error: Password must contain at least one number"

  IF password does not contain any uppercase letter (A through Z):

  RETURN "Valid"

Key rules for your answers:

- Only return "Valid" at the very end after all checks pass
